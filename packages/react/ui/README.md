# Casual UI React

基于React17+的一个随意的组建库

## 安装

```bash
# 使用NPM
npm install casual-ui-react

# 使用YARN
yarn install casual-ui-react

# 使用PNPM
pnpm install casual-ui-react
```

## 使用

```jsx
import 'casual-ui-react/dist/style.css'
import { CButton } from 'casual-ui-react'

const APP = () => <CButton label="按钮" />
```
